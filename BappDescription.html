Many web sites compress resources such as JavaScript files in order to increase the loading speed. This extension makes it possible to decompress and beautify most resources, to facilitate testing.

Features:

  • Manual beautifying of requests and responses
  • Automatic beautifying of responses in the Proxy tool
  • Automatic beautifying of responses in the message editor
  • Can support Burp's target scope configuration
  • Mimicks the exact behaviour of JSBeautifier.org website by using Rhino library
  • Supports multiple file types (JS, CSS, HTML, etc.)
  • Detects packers and obfuscators (based on JSBeautifier.org)
  • Highlights syntax highlighter in the read-only editor by using Fifesoft RSyntaxTextArea library

This extension is based on the following modules/libraries:

  • JS files of http://jsbeautifier.org/ - Written by Einar Lielmanis, einar@jsbeautifier.org
  • Rhino library - http://www.mozilla.org/rhino/
  • Fifesoft RSyntaxTextArea library: http://fifesoft.com/rsyntaxtextarea/

Limitations:

  • Only supports UTF-8 for texts
  • Requires Java version 7
